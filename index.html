Bases del desarrollo Front-end

En el amplio mundo del desarrollo web existen pilares fundamentales para la creacion de paginas online funcionales, dichos pilares se conocen como :

- Html
- Css
- Javascript


Javascript

Es conocido como un lenguaje de programacion utilizado en documentos HTML principalmente para añadir caracteristicas interactivas a un Sitio Web. Fue inventado por Brendan Eich, co-fundador del proyecto Mozilla Foundation y la Corporacin Mozilla. Logrando de esta forma uno de los lenguajes de programacion mas populares del mundo.

Html

Es conocido como un lenguaje de marcado utilizado en la creacion de paginas web. Se trata de las siglas que corresponden a HyperTextMarkupLanguage, es decir, un Lenguaje de Marcas de Hipertexto

Css

Conocido como un lenguaje de diseño grafico capaz de definir y crear la presentacion de un documento estructurado escrito en un lenguaje de marcado(HTML),es decir, tiene la capacidad de proporcionar o cambiar las caracteristicas de algun elemento dentro de una pagina web como darle color a un texto o hacerlo mas grande.

Frameworks

Ademas de los grandes pilares del desarrollo web mencionados debemos destacar la importancia y el poder de los Frameworks para cada una de las tecnologias de hoy en dia.

Al hablar de Frameworks puede definirse como una variedad de herramientas especializadas, con el objetivo de mejorar la eficiencia y efectividad al momento de realizar codigo

Entre las Tecnologias mas importante se destacan:

-Javascript:
Angular,Reach,VueJs

-Css:
Bootstrap
